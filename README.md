# ✨ What is Monsken?

Monsken is a local-first emotional operating system designed for reflection, memory flow, and ambient awareness.

It is not built around:

* productivity metrics
* streak systems
* optimization loops
* notifications
* performance anxiety

Instead, Monsken treats digital space as a form of emotional architecture.

The interface behaves less like software and more like:

* moonlight drifting through fog
* memories flowing through water
* emotional weather changing over time

Monsken is designed especially for:

* reflective thinkers
* emotionally sensitive users
* ADHD-friendly low-pressure interaction
* people exhausted by traditional productivity software

---
